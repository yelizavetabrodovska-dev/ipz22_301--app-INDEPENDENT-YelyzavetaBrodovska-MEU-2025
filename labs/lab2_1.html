Завдання №2

Засобами HTML та CSS виконати Завдання №2, а саме створити окремі веб-сторінки:

1. Розмітка за допомогою таблиці

  • Фіксована таблична верстка
  • Гумова таблична верстка

(Номер варіанта – остання цифра у списку групи)

2. Розмітка за допомогою блоків (плаваючих елементів)

  • Фіксована блокова верстка
  • Гумова блокова верстка

(Номер варіанта – остання цифра у списку групи + 1)

Фіксована таблична верстка

<!DOCTYPE html>
<html lang="uk">
<head>
<meta charset="UTF-8">
<title>Фіксована таблична верстка</title>
<style>
  table {
    width: 600px;
    height: 400px;
    border-collapse: collapse;
    margin: 20px auto;
  }
  td {
    border: 2px solid black;
    text-align: center;
    font-size: 24px;
  }
  .t1, .t3 { background: #007BFF; color: white; }
  .t2, .t5 { background: #FFC107; }
  .t4 { background: white; }
</style>
</head>
<body>
  <table>
    <tr>
      <td colspan="3" class="t1">1</td>
    </tr>
    <tr>
      <td class="t5">5</td>
      <td class="t4">4</td>
      <td class="t2">2</td>
    </tr>
    <tr>
      <td colspan="3" class="t3">3</td>
    </tr>
  </table>
</body>
</html>

Гумова таблична верстка

<!DOCTYPE html>
<html lang="uk">
<head>
<meta charset="UTF-8">
<title>Гумова таблична верстка</title>
<style>
  table {
    width: 80%;
    height: 80vh;
    border-collapse: collapse;
    margin: 20px auto;
  }
  td {
    border: 2px solid black;
    text-align: center;
    font-size: 2vw;
  }
  .t1, .t3 { background: #007BFF; color: white; }
  .t2, .t5 { background: #FFC107; }
  .t4 { background: white; }
</style>
</head>
<body>
  <table>
    <tr>
      <td colspan="3" class="t1">1</td>
    </tr>
    <tr>
      <td class="t5">5</td>
      <td class="t4">4</td>
      <td class="t2">2</td>
    </tr>
    <tr>
      <td colspan="3" class="t3">3</td>
    </tr>
  </table>
</body>
</html>

Фіксована блокова верстка

<!DOCTYPE html>
<html lang="uk">
<head>
<meta charset="UTF-8">
<title>Фіксована блокова верстка</title>
<style>
  body {
    width: 600px;
    margin: 20px auto;
  }
  .header { background: #FFC107; height: 80px; }
  .sidebar { background: #007BFF; width: 120px; height: 300px; float: left; }
  .content { background: white; width: 240px; height: 200px; float: left; }
  .aside { background: #007BFF; width: 120px; height: 200px; float: left; }
  .top { background: red; width: 360px; height: 50px; float: left; }
  .footer { background: #FFC107; height: 50px; clear: both; }
</style>
</head>
<body>
  <div class="header">1</div>
  <div class="sidebar">2</div>
  <div class="top">Top</div>
  <div class="content">3</div>
  <div class="aside">4</div>
  <div class="footer">5</div>
</body>
</html>

Гумова блокова верстка

<!DOCTYPE html>
<html lang="uk">
<head>
<meta charset="UTF-8">
<title>Гумова блокова верстка</title>
<style>
  body {
    width: 90%;
    margin: 20px auto;
  }
  .header { background: #FFC107; height: 10vh; }
  .sidebar { background: #007BFF; width: 20%; height: 60vh; float: left; }
  .top { background: red; width: 60%; height: 10vh; float: left; }
  .content { background: white; width: 40%; height: 40vh; float: left; }
  .aside { background: #007BFF; width: 20%; height: 40vh; float: left; }
  .footer { background: #FFC107; height: 10vh; clear: both; }
</style>
</head>
<body>
  <div class="header">1</div>
  <div class="sidebar">2</div>
  <div class="top">Top</div>
  <div class="content">3</div>
  <div class="aside">4</div>
  <div class="footer">5</div>
</body>
</html>
